## Test Commands
- Run all tests: `pytest`
- Run single test: `pytest path/to/test_file.py::TestClass::test_method`

## Code Style Guidelines
- Use type hints for all function parameters and return types.
- Follow PEP8: CamelCase for classes, snake_case for functions/variables.
- Imports: standard library first, then third-party, then local modules.
- Use f-strings for string formatting.
- Error handling: raise descriptive ValueError or AttributeError exceptions.
- Initialize lists/dicts as []/{} in __init__ instead of None.
- Add __repr__ methods to classes for debugging.
- Avoid comments unless necessary; code should be self-explanatory.</content>
